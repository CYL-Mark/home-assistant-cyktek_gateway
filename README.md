# home-assistant-cyktek_gateway
cyktek_gateway integration for Home Assistant

A custom component to handle [CYLTek](https://www.cyl-tek.com) IOT devices in [Home Assistant](https://home-assistant.io).

## Install
1. Ensure Home Assistant is updated to version 2021.4.0 or newer.
2. Download it and manually move to the `custom_components` folder and restart the Home Assistant.
3. Once the integration is installed follow the steps, go to setup > integration and search for `cyltek_gateway`.
4. Enjoy.
